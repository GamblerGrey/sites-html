Html


HyperText Markup Language ou, Html é uma liguagem de marcação que utilizamos para criar e exibir conteudo na web.

  • Tags principais do Html
  • Algumas complementares
  • Considerações finais


Tags principais do html

As tags principais são usadas em todos projeto base para um site, blog e criações diversas.
Como se diz o limite a imaginação.


    Html - tag é usada para definir o início do documento HTML.


    Head - Esta tag é usada para incluir informações sobre a página, como o título da página, meta tags e scripts.


    title - Esta tag é usada para definir o título da página, que aparece na barra de título do navegador.


    body - Esta tag é usada para definir o conteúdo da página, como textos, imagens, vídeos, links e outros elementos.


    h1 até h6 - Essas tags são usadas para criar títulos e subtítulos, com diferentes níveis de importância.


    p - Esta tag é usada para criar parágrafos de texto


    a - Esta tag é usada para criar links para outras páginas da web ou para arquivos no próprio site.


    img - Esta tag é usada para inserir imagens na página.


Algumas complementares


tags que seu uso são simples ou, não muito recorrentes como br


br - Essa tag tem a função de quebra linha tanto em tags como em espaços


Strong - A tag que usamos para deixa em negrito as palavras que queremos destacar


Mark - A tag que tem a função mais que simples que é funcionar como uma marcador de texto


Considerações Finais


É liguagem bem vasta porém tem suas limitações como não pode estiilzar a pagina sem o css.
Entretanto, as diversas opções de customiza de texto são impressionates por exemplo botões, toggle, divs e outros que
nos permite cria diversos objeto que serve de esqueleto para css
